### Documentation- For Onboarding but affects this group
      1. Docs have atrophied from "among the best" to "behind the curve"
      2. First-party onboarding guide a la "the Rust Book"/"the Go Book"
         1. I have a prototype for a "Tour of Go" equivalent — it doesn't look nice, but it gives a hint. Content writing requires the skills of an educator though: https://github.com/derickr/php-tour
      3. "PHP has the strongest type system of any interpreted language."
      4. Installing Composer/FrankenPHP out of the box

### Promoting the community — PHP's strongest factor
      1. Anyone who starts using PHP (or any tool built in PHP — Drupal, WP, Laravel, etc.) they are now a PHP developer and part of the community
      2. We all need people, especially under-represented groups (i.e. not men)
      3. Talk PHP up without talking others down
