# The PHP Ambassador Program

## Overview
The PHP Ambassador Program is one of the Special Interest Groups facilitated by The PHP Foundation. 

### Mission
This group aims to improve the perception of PHP in open source spaces, outside the PHP bubble. The specific goals and details of this group's activities are TBD.

### How to Join

We will be hosting a kick-off meeting for this group on **Tuesday, June 20, 2026 at 10:30 am EDT / 2:30 pm UTC.** If you are interested in attending this meeting, please [fill out this form](https://forms.gle/kM7es67SgVPtgkze7) to register your interest. If you cannot attend this meeting but you are interested in participating in this group asynchronously, you can watch this repo for updates.
