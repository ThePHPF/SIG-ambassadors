# The PHP Ambassador Program

## Overview
The PHP Ambassador Program is one of the Special Interest Groups facilitated by The PHP Foundation. 

### Mission
This group aims to improve the perception of PHP in open source spaces, outside the PHP bubble. The specific goals and details of this group's activities are TBD.

### How to Join

We have broken this group into 3 sub-groups to allow those with specific interests to contribute easier:

* **Research**: This group finds and curates data about PHP usage or other relevant data points that can be used by the community in outreach, articles, presentations. ([Discord](https://discord.com/channels/356354025865740288/1521542949782159390), [Docs Folder](https://github.com/ThePHPF/SIG-ambassadors/tree/main/subgroup-research), [Discussions](https://github.com/ThePHPF/SIG-ambassadors/discussions/categories/research-sub-group))
* **Speakers**: This group is for speakers who want to give talks outside the PHP space. Shares CfPs, helps each other with abstracts, etc.([Discord](https://discord.com/channels/356354025865740288/1521543086218678352), [Docs Folder](https://github.com/ThePHPF/SIG-ambassadors/discussions/categories/speakers-sub-group), [Discussions](https://github.com/ThePHPF/SIG-ambassadors/discussions/categories/marketing-sub-group))
* **Marketing**: This group helps create design assets, collects testimonials and stories for “why PHP” that can be used on the php.net website or other places, helps refine the messaging we can all use.([Discord](https://discord.com/channels/356354025865740288/1521543396865474571), [Docs Folder](https://github.com/ThePHPF/SIG-ambassadors/tree/main/subgroup-marketing), [Discussions](https://github.com/ThePHPF/SIG-ambassadors/discussions/categories/marketing-sub-group))

_You are welcome to participate in any or all of them._

**There are a few things you can do to start participating immediately:**

* Head to the SIG-Ambassador Discussions and [introduce yourself](https://github.com/ThePHPF/SIG-ambassadors/discussions/categories/introductions)
* Weigh in on both a general [time](https://github.com/ThePHPF/SIG-ambassadors/discussions/5) and [day](https://github.com/ThePHPF/SIG-ambassadors/discussions/6) that might work for you for the next meeting
* Start some [discussions](https://github.com/ThePHPF/SIG-ambassadors/discussions) in the area that you're interested in! I know you all have lots to say ☺️ Share your ideas! 
* [Join phpc Discord](https://discord.phpc.social/) if you aren't already there. (Special thanks to the folks hosting and moderating the phpc Discord who are kindly providing infrastructure for us to have a place to chat directly with each other)


### Code of Conduct
By participating in this Special Interest Group, you are agreeing to adhere to [The PHP Foundation Code of Conduct](https://thephp.foundation/code-of-conduct/), so make sure you have read and understand our expectations for behavior in this space. :heart:
